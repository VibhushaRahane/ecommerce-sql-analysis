# E-commerce Sales Analysis (SQL Project)

## 📌 Objective
Analyze e-commerce data to extract meaningful insights on customer behavior, product performance, and overall sales trends.

## 🛠 Tools Used
- MySQL

## 📂 Database Structure
The database consists of 4 tables:
- Customers
- Orders
- Products
- Order_Items

## 🔍 Key Analysis Performed
- Total revenue calculation
- Top-selling products
- Customer order analysis
- Revenue by city
- Monthly order trends

## 🧠 SQL Concepts Used
- JOINs (INNER JOIN, LEFT JOIN)
- GROUP BY & Aggregations (SUM, COUNT)
- Subqueries
- Views
- Indexing

## 📊 Key Insights
- Identified high-value customers and top-performing products
- Analyzed sales trends across different cities and months
- Improved query performance using indexing

## 📁 Files
- ecommerce_database.sql (contains database creation, data insertion, and analysis queries)
